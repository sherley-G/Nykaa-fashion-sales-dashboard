# Nykaa sales dashboard
This project analyzes Nykaa sales data to identify customer purchasing behavior, regional sales performance, product trends, and overall business insights using interactive Excel dashboards.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning Techniques

## Workbook Contents
- Raw Dataset
- Cleaned Data
- Pivot Table Analysis
- KPI Metrics
- Interactive Dashboard
- Business Insights

## Key Insights
- Identified top-performing product categories
- Analyzed regional sales distribution
- Evaluated customer purchase patterns
- Compared sales trends across categories
- Visualized KPIs using interactive dashboards
